# Clark Kitchen - Portfolio

Personal portfolio website built with Next.js and the Batman Beyond design system.

## Links

- **Live Site**: [clarkkitchen.com](https://clarkkitchen.com)
- **Source Code**: [GitHub](https://github.com/goldbar123467/clarkkitchen.com)

## Tech Stack

- Next.js 14+
- React 18+
- Tailwind CSS
- shadcn/ui
- TypeScript

## Getting Started

```bash
bun install
bun dev
```

Open [http://localhost:3000](http://localhost:3000) to view.
